# Netflix-Movies-Data-Analysis
Exploratory Data Analysis of Netflix Movies using Python, Pandas, Matplotlib and Seaborn.

## Key Questions Explored

- What is the most frequent genre?
- Which genre receives the highest number of votes?
- Which movie has the highest popularity?
- Which movie has the lowest popularity?
- Which year released the most movies?

## Dataset
Netflix Movies Dataset

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
